---
description: 게이트의 주기 조건을 무시하고 OKF 배치를 지금 즉시 강제 실행한다(락은 존중).
---

지금 즉시 OKF 배치를 강제 실행하라. `batch_interval_hours` 주기 게이트는 무시하되(이 커맨드가
직접 배치 드라이버를 실행하므로 애초에 주기 검사를 거치지 않는다), 배치 자체의 락(`batch.lock`)은
존중한다 — 이미 다른 배치가 실행 중이면 드라이버가 스스로 즉시 종료한다.

Bash로 아래를 실행하라:

```
node "${CLAUDE_PLUGIN_ROOT}/bin/batch.mjs"
```

- 이 스크립트는 자체적으로 `OKF_HOME`을 해석하고 필요한 락/sweep/청크 처리를 전부 수행한다 —
  추가 인자나 환경변수는 필요 없다.
- 실행이 끝나면 종료 코드와 출력(있다면)을 사용자에게 그대로 보고하라. 실패(0이 아닌 종료
  코드)면 무엇이 실패했는지 출력 내용을 근거로 요약하라.
- 실행 시간이 길 수 있다(청크당 최대 15분 타임아웃 × 청크 수) — 오래 걸려도 정상이니 중간에
  임의로 중단시키지 마라.
- 이미 배치가 실행 중이어서 락 때문에 즉시 종료된 경우, 그 사실을 사용자에게 안내하고
  `/okf-status`로 진행 상황을 확인해보라고 제안하라.
